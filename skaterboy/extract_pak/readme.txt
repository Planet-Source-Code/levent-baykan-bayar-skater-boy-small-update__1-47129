pakextract utility v1.0
=======================
can be used to extract pak files.Just rename pakextr.bin to pakextr.exe

usage:
=======================
pakextract <pakfilename>

all files will be extracted with 'file' prefix + index number + extension of file,such as file1.bmp or file45.wav,and a filenames.txt showing real names of files.

sorry no directory reconstruction in this version. 

pakextr.pas is pascal source code,that can be compiled with gnu pascal compiler,or Delphi
